# Council of Alphas - Dashboard

React + Vite + Tailwind CSS dashboard for visualizing pipeline results.

**Live**: [council-of-alphas.vercel.app](https://council-of-alphas.vercel.app)

## Tabs

1. **Pipeline Overview** - Architecture, config, stages
2. **Lineage Tree** - 12 strategies -> 4 champions -> 3 hybrids -> ranked survivors
3. **Regime Filters** - Session/trend/volatility classification, 3D buckets, routing table
4. **Strategy Tear Sheet** - Equity curve, drawdown, KPIs, regime diagnostics per hybrid
5. **Leaderboard** - Podium, comparison table, strategy details
6. **Correlation Matrix** - PnL correlation between profitable hybrids

## Dev

```bash
npm install
npm run dev
```

## Data

`src/data.js` is auto-generated by `generate_data_js.py` from pipeline results. Do not edit manually.
